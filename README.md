## Getting Started

First, download nodejs version > 8, https://nodejs.org/en/

Run the command

```bash
npm install
```

to install all the required dependancies, then to run the development server:

```bash
npm run dev
# or
yarn dev
```

Open http://localhost:3000 with your browser to see the result.
